# Pre-requisites
Node.js version 18.18.0 or newer is required. You can download it from Node.js Official Website.
Yarn version 1.22.19 is required. You can install it using the following command: npm install -g yarn


# Installation
Clone the repository: git clone https://gitlab.duthu.net/S52000668/nodejs-502070-2324-final-project
Change to the project directory: cd nodejs-502070-2324-final-project/
Install project dependencies: yarn
Build and run the project: yarn dev
Open a web browser and navigate to http://localhost:8080 to access the local development version of the website.


# Deployment
To deploy your changes to the server (onrender.com):
Make code changes in the nodejs-502070-2324-final-project folder.
Push your code to the main branch, and the server onrender.com will automatically deploy the updated code.
Access the deployed website at http://tech-hut.onrender.com/.
